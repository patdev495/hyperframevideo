# Hyperframe Video

Local-first **News-to-Video Pipeline** for turning news sources into short video production artifacts for HyperFrames.

## Local setup

Install dependencies and run tests with `uv`:

```powershell
uv run pytest
```

Run the CLI:

```powershell
uv run hyperframe-video --help
```
