# Local-first execution with HyperFrames rendering

The project will run the MVP locally: orchestration, file artifacts, HyperFrames composition generation, local TTS where available, and MP4 rendering happen on the user's machine. HyperFrames is the rendering layer, while this project owns news discovery, source reading, script approval, voiceover provider selection, production-run artifacts, and orchestration; internet access may still be used for current news discovery, so this is local-first execution rather than offline-only operation.
